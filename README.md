# Comparative Cancer Resistance Across Species

This project explores evolutionary mechanisms associated with cancer resistance in organisms including:

- Humans
- Elephants
- Bowhead whales
- Tardigrades

Topics explored:

- TP53 expansion
- DNA repair systems
- Comparative oncology
- Translational biomedical research
- Evolutionary medicine

## Technologies Used

- Python
- Pandas
- Matplotlib
- Google Colab
